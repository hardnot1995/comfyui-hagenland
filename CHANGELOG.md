# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-04-11
### Added
- Initial release (Plan v1) of the ComfyUI Video Sequence Controller node.
- Features:
  - Iterative processing of video segments.
  - Unique project folder creation using project name and random number.
  - Reset functionality to clear progress and restart the workflow.
  - Configuration options: segment count (default 10) and frames per segment (default 81).
  - Grouping support via NODE_GROUP and an icon in NODE_NAME.
